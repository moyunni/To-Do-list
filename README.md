# To-Do-list 📝

Современное Android-приложение для управления задачами, написанное на **Kotlin** с использованием **Jetpack Compose**. Основной фокус проекта — реактивность интерфейса и плавный пользовательский опыт.

## 🚀 Стек технологий
* **UI**: Jetpack Compose (Material You / Material 3)
* **Architecture**: MVVM + UDF (Unidirectional Data Flow)
* **Database**: Room Persistence Library
* **State Management**: Kotlin Flow & LiveData

## ✨ Ключевые особенности
* **Реактивность**: Интерфейс мгновенно подстраивается под изменения данных в базе.
* **Плавность (60 FPS)**: Оптимизированные анимации и отсутствие задержек при рекомпозиции.
* **Material You**: Поддержка динамических тем оформления (светлая/темная/системная).
* **Приоритеты**: Цветовая индикация важности задач (Low, Medium, Critical).
* **Поиск**: Быстрая фильтрация списка дел в реальном времени.
